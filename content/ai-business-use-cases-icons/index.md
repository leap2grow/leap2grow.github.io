    if (hexColor.length === 3) { hexColor = hexColor.split('').map(function (hex) { return hex + hex;}).join(''); }

  // Get contrast color;
  function getColorContrast(hexColor) {
    const rgb = hexToRgb(hexColor);
    let yiq = ((rgb[0] * 299) + (rgb[1] * 587) + (rgb[2] * 114)) / 1000;
    const colorContrast = (yiq >= 128) ? 'hsl(0 0% 0%)' : 'hsl(0 0% 100%)';

    // return
    return colorContrast
  };

  function hexToHsl(hexColor) {
    const rgb = hexToRgb(hexColor);

    // Make r, g, and b fractions of 1
    const r = rgb[0]/255;
    const g = rgb[1]/255;
    const b = rgb[2]/255;

    const max = Math.max(r, g, b), min = Math.min(r, g, b);
    let h, s, l = (max + min)  /  2;  
    if ( max == min ) { h = s = 0; } else {
      let d = max - min;
      s = l > 0.5 ? d / (2 - max - min) : d / (max + min);
      switch(max){
        case r: h = (g - b) / d + (g < b ? 6 : 0); break;
        case g: h = (b - r) / d + 2; break;
        case b: h = (r - g) / d + 4; break;
      }
      h /= 6;
    }

    return [Math.round(h * 360),Math.round(s * 100),Math.round(l * 100)]
  }

      <p>AI is changing the world - and the economy. Software now <a href="https://1secondcopy.com/?ref=nicksaraev.com">writes</a>, <a href="https://1secondpainting.com/?ref=nicksaraev.com">draws</a>, and works thousands of times faster than the average human, and most industries are ripe for the picking.</p><p>Naturally, this represents an opportunity. Big businesses and regulatory agencies are going to take years to figure this out. In the meantime, <em>you</em> can take advantage of the gap in the market to both improve your business, make more money, and make your life easier.</p><p>In this series (which I'm also turning into a newsletter), the goal is to jam-pack every volume with simple, real ways to apply AI to various industries. I include examples, and show you specific steps that you can take to multiply your effectiveness. I also take industry requests.</p><p>So, without further fluff: let's get into how to make an AI icon business.</p><h2 id="the-emerging-artificial-intelligence-icon-industry">The emerging artificial intelligence icon industry</h2><p>Right now, icons are laboriously hand-drawn by designers. If used widely, they're refined and disseminated in the next big 'icon package' and hosted on an icon website - think <a href="https://mui.com/components/material-icons/?ref=nicksaraev.com">Material UI</a>, <a href="https://icons8.com/?ref=nicksaraev.com">icons8</a> or <a href="https://www.flaticon.com/?ref=nicksaraev.com">flaticon</a>. </p><p>This takes a bunch of time and money. Fully custom icon designs can cost between $20 and $40; that's a fair chunk of change for something no bigger than your pinky.</p><p>You could easily bootstrap an artificial intelligence icon startup to dominate in this space. Alternatively, if you're not willing to do the heavy lifting, you can sell the bulk of your icons to the big players in the market (I mentioned a few of them earlier), or just generate artificial intelligence icons for your personal business or project.</p><p>Here's how.</p><h2 id="how-to-create-an-artificial-intelligence-icon-startup">How to create an artificial intelligence icon startup</h2><p>AI icon startups are easy. First, you need a reliable way to create the product.</p><h3 id="1-create-ai-icons-using-diffusion">1. Create AI icons using diffusion</h3><p>The best way to create icons with artificial intelligence is <strong>diffusion</strong>. This is the statistical process that the majority of AI art is now created with. </p><p>You don't need to be an AI or programming expert to use diffusion reliably - you just need to understand how to <em>find</em> new projects and <em>use tools that have already been created for you</em>, like <a href="https://colab.research.google.com/?utm_source=scs-index">Colab</a><em>.</em></p><p>Many diffusion models exist, but the one most primed for AI icons is probably Katherine Crowson's <a href="https://colab.research.google.com/drive/1TBo4saFn1BCSfgXsmREFrUl3zSQFg6CC?ref=nicksaraev.com">Text to Image with Classifier Free Guidance</a>.</p><p>Using a prompt like <em><strong>user interface icon, vector, trending on artstation, opposed symmetry | opposed symmetry, user interface icon</strong></em>, you can get something like this in 15 seconds:</p><figure class="kg-card kg-image-card"><img src="https://leap2grow.github.io/content/images/2022/03/download--2-.png" class="kg-image" alt loading="lazy" width="512" height="512"></figure><p>Not bad for your first go. Spend a few hours testing for the prompts that work best for your particular use case and you can significantly improve the quality.</p><h3 id="2-create-a-post-processing-stack-to-standardize-sizes-imagery">2. Create a post-processing stack to standardize sizes &amp; imagery</h3><p>Now it's time to turn your crude artificial intelligence icon set into something polished using post-processing. </p><p>Here, you can hire a designer to cut through the center of each icon &amp; mirror it (for simplicity and to standardize symmetry), and then crop and export each icon in the 4 standard sizes: 16px, 32px, 64px, and 128px. Assuming a manual (but optimized) process, you'd be looking at between thirty seconds to a minute per icon, or perhaps thirty icons for an hour of your designer's time. You can also do this manually to start.</p><p>During this time, deal with any watermark-type-entities that come up (which are common when attempting to create 'vector' images using AI - look at the last image carefully).</p><p>Alternatively, you could post-process using Photoshop Batch Actions or an equivalent (below are a few bullet point actions you can take, but I won't get into this further):</p><ul><li>Center the top-left, top-right, bottom-left, or bottom-right quadrant</li><li>Cut halfway through the icon</li><li>Mirror the result</li><li>Crank up contrast/brightness to a pre-ordained level to eliminate the majority of noise</li></ul><p>Here's an example of what that might look like for one of the icons we generated earlier:</p><figure class="kg-card kg-image-card"><img src="https://leap2grow.github.io/content/images/2022/03/image-5.png" class="kg-image" alt loading="lazy" width="508" height="509"></figure><p>Pretty neat, huh? It's easy to imagine using this in a logogram or logo (I could see this on Discord or something similar).</p><h3 id="3-host-ai-icons-using-marketplace-website-software-or-a-pre-created-platform">3. Host AI icons using marketplace website software, or a pre-created platform</h3><p>Building stuff yourself takes time. Why spend months creating a platform to validate an idea when a few days will do?</p><p>Use a premade solution like WooCommerce, Shopify, or an equivalent to start hosting your icons. You can hire a virtual assistant for between $5USD/hr and $10USD/hr using something like <a href="https://www.onlinejobs.ph/?ref=nicksaraev.com">onlinejobs.ph</a> to manage and upload your icon set, and if you'd like to stretch your budget further, you can even have them do some of the graphic design we mentioned earlier.</p><p>If you don't have any management experience, stick with something simple:</p><ul><li>Create a daily task list you'd like them to fulfill.</li><li>For example: uploading fifty icons per day, responding to customer inquiries, and thirty minutes of looking for icon commission requests on <a href="https://reddit.com/r/icons?ref=nicksaraev.com">reddit.com/r/icons</a>.</li><li>For each item, write a detailed, step-by-step procedure they can follow to complete the task.</li><li>For the first week, check in with them at the end of every day. After the first week, check in with them whenever suits your schedule.</li></ul><h3 id="4-market-your-ai-icons-on-twitter-or-other-social-media-platforms">4. Market your AI icons on Twitter or other social media platforms</h3><p>Artificial intelligence art is blowing up right now. You could easily carve out a niche with an 'AI icon' account on Twitter. Here's how:</p><ul><li>On a day when you have the time, pre-generate a few hundred icons across a wide variety of sectors/interests.</li><li>Follow a few dozen AI art accounts on Twitter, and comment on their posts consistently. Optimize your profile. Link back to your new website, and make it clear what you do.</li><li>Several times per day, tweet one of your icons with the requisite hashtags. There are several great approaches to this that you can emulate (be different, of course): <a href="https://twitter.com/nearcyan?ref=nicksaraev.com">@nearcyan</a>, for example, does this very well.</li></ul><h3 id="creating-an-ai-icon-business-conclusion">Creating an AI icon business: conclusion</h3><p>In summary, there currently exists a gap in the market that you can take advantage of: AI icons. With a bit of clever prompting, a good post-processing stack, and a reliable assistant, a business like this can be grown in an automated fashion with little to no personal involvement.</p><p>You can also extend this approach to other, related B2B services: AI logo making, AI branding, and so on. But be warned - these opportunities won't exist forever. All it takes is one major player to rise up to the challenge in the next few years, and it'll be gone.</p>

      });

      let subMenu, hasItems
      navItems.forEach((item, index) => {
        if (item.classList.contains('is-subitem') && !navItems[index - 1].classList.contains('is-subitem')) {
          navItems[index - 1].classList.add('is-mainitem'); 
          navItems[index - 1].setAttribute('x-on:click.outside', 'closeSubmenus(event)')
        }
        subMenu = item.classList.contains('is-subitem') ? subMenu : document.createElement('ul');
        if (item.classList.contains('is-subitem')) { 
          subMenu.appendChild(item)
          subMenu.setAttribute('data-submenu', '')
          subMenu.getAttribute('id') === null ? subMenu.setAttribute('id', navItems[index - 1].getAttribute('data-slug')) : ''
          const menuType = subMenu.closest('nav').getAttribute('data-nav')
          subMenu.classList = menuType === 'desktop' ? desktopClass : mobileClass
        } else { 
          item.appendChild(subMenu)
        }
      });

      const dropdownMenus = document.querySelectorAll('[data-nav] .is-mainitem')
      const toggle = document.querySelector('[data-toggle-template]');

      dropdownMenus.forEach(menu => { 
        const toggleBtn = toggle.content.firstElementChild.cloneNode(true);
        toggleBtn.setAttribute('aria-controls', menu.getAttribute('data-slug'))
        menu.insertBefore(toggleBtn, menu.children[1]);
      });
