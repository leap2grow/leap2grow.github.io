  function getColorContrast(hexColor) {
    const rgb = hexToRgb(hexColor);
    let yiq = ((rgb[0] * 299) + (rgb[1] * 587) + (rgb[2] * 114)) / 1000;
    const colorContrast = (yiq >= 128) ? 'hsl(0 0% 0%)' : 'hsl(0 0% 100%)';

  function hexToHsl(hexColor) {
    const rgb = hexToRgb(hexColor);

    // Make r, g, and b fractions of 1
    const r = rgb[0]/255;
    const g = rgb[1]/255;
    const b = rgb[2]/255;

    const max = Math.max(r, g, b), min = Math.min(r, g, b);
    let h, s, l = (max + min)  /  2;  
    if ( max == min ) { h = s = 0; } else {
      let d = max - min;
      s = l > 0.5 ? d / (2 - max - min) : d / (max + min);
      switch(max){
        case r: h = (g - b) / d + (g < b ? 6 : 0); break;
        case g: h = (b - r) / d + 2; break;
        case b: h = (r - g) / d + 4; break;
      }
      h /= 6;
    }

    return [Math.round(h * 360),Math.round(s * 100),Math.round(l * 100)]
  }

Who could pass this up?</p><p>Ackerman bargaining is a way to streamline the negotiation process to ensure consistency and empathy. It also makes use of a curious mathematical (and psychological) phenomenon to get you the price you want. It's probably the most succesful bargaining method of all time, and the top Craigslist negotiators, finance deal closers, and lawyers use it all the time to get what they want</p><p>Here‚Äôs how Ackerman Bargaining works:</p><ol><li>Set a target price (one that you‚Äôre willing to buy or sell at)</li><li>Set your first offer at <em>65%</em> of that price (if you‚Äôre buying), or <strong>135%</strong> of that price (if you‚Äôre selling)</li><li>Move your offer in three increments: 85% (or <strong>115%</strong>), 95% (or <strong>105%</strong>), and then finally 100%. (if you're stuck with crazy decimal points, round them off to the nearest 1).</li><li>Between each movement, be as empathetic as possible in how you say ‚Äúno‚Äù. Get them in your shoes‚Ää‚Äî‚Äätell them about how it makes you feel, and how much you think the product is really worth. Never dismiss their counter-offers emotionally.</li><li>On the final offer, don‚Äôt give them a round number like $5,000. Change the last few digits to something seemingly concrete, like $5,032. This helps imply that you‚Äôre at your limit.</li><li>If they don‚Äôt budge, try throwing in a non-monetary item alongside your final offer to imply that you‚Äôve maximized your budget. Ideally this would be something they‚Äôve expressed interest in during the negotiation process that you don‚Äôt really care about, but it's up to you.</li></ol><p>Whew, that was a lot of numbers. To make things more concrete, here's a simple selling example that demonstrates all six of the above principles (ideally, you would do all of the calculations beforehand).</p><h3 id="ackerman-bargaining-example">Ackerman Bargaining Example</h3><p>Let's say you want to buy a smartphone that's currently listed at $100. You would like to buy the smartphone for $80 instead of $100, since that's what you think it's worth. Using the Ackerman bargaining method, you set your initial price at:</p><pre><code class="language-text">0.65 * $80 = $52
</code></pre><p>You tell the other party that you're very interested in the phone, and you think it's a great quality product, but you can only do $52. Naturally, at this point, they're probably thinking "there's no way in hell I'm doing that! This is worth at least $90", and that's fine. They'll probably tell you as much right away.</p><p>You then calculate the next step on the ladder:</p><pre><code class="language-text">0.85 * $80 = $68
</code></pre><p>So you bring this to the seller, and let them know that it really is <em>such</em> a nice phone, and you're <em>so</em> very sorry, but $68 is the most that you can do. Really.</p><p>At this point, either they accept your price, they stand firm at $90, or they make a small concession (to let's say $85). Any of those three options are just dandy‚Äîbecause remember, you still have a couple of steps to go. You repeat this two more times:</p><pre><code class="language-text">0.95 * $80 = $76
1.00 * $80 + $0.32 (remember, last offers are non-round numbers ) = $80.32
</code></pre><p>In all likelihood, if you've validated the product and built rapport with the seller correctly, they have probably already said yes. But if not, giving them a seemingly carefully-calculated figure like $80.32 makes it seem like you're at your limit, and significantly increases the likelihood that they close the deal.</p><p>Finally, if they're <em>still</em> holding out, your last option is to give them a non-monetary concession of some kind. So your final offer could be something like:</p><blockquote>I am sorry, I just can't give you any more than $80.32. However, if it makes it more convenient for you, I would be willing to drive out to pick it up to save you some time. How does that sound?</blockquote><p>And that's it. Copy and paste this method to any product you want to buy or sell from here on out, and you'll probably improve your outcomes by something like 30% at the <em>minimum</em>.</p><p>The Ackerman bargaining method works. It‚Äôs a delicate play on human psychology, and the real beauty of using it is how you get to build positive rapport while negotiating. Unlike most bargaining methods, it also doesn‚Äôt make the other party feel like they‚Äôve been ripped off or somehow cheated out of a price. This allows you to continue to negotiate on Craigslist for other products with the same people in the future.</p><hr><p>Of course, there <em>are</em> some methods that, while they may help you strike a better deal, definitely <em>do</em> make the other party feel like they've been ripped off. Accordingly, you do not want to use them more than once on the same person, for fear of safety and reputation.</p><p>These methods are often referred to as the 'dark arts of bargaining on Craigslist'. I'm going to talk about one in particular, but I am not joking when I say this: <strong>be careful applying this method to people that you interact with frequently</strong>. You can wind up in some serious trouble, both personally and professionally.</p><p>Again: this is only to be used if you absolutely must. But if you're curious (I know you are, you dirty bargain-hunter), read on.</p><h2 id="use-price-anchoring-with-multiple-pitches-to-fabricate-a-good-deal-warning-unethical"><strong>Use Price Anchoring With Multiple Pitches To Fabricate A Good Deal (warning: unethical)</strong></h2><p><strong>The last, and most unethical method on how to negotiate on Craigslist is optional</strong>, and, for many, a little too far. I include it here for completeness‚Äô sake. It involves sending messages from more than one email or phone number to psychologically anchor the price in the prospect‚Äôs mind.</p><p>As someone looking to sell a product on Craigslist, receiving a single, lonely low-ball offer is usually easy to get over. But when you start getting two, three, or four in quick succession, it can make you think that perhaps their product really <em>isn‚Äôt</em> worth the money you're trying to get for it‚Äîand it makes you a lot more suggestive to haggles than you would have been otherwise.</p><p>This method is relatively simple: pitch a random distribution of offers 40%-50% lower than their listed price. If they want $100, for example, send three emails from different email addresses offering $50, $60, and $55. You then send a fourth offer (your <em>real</em> offer) from your actual email asking for, let's say, $70. At this point, they‚Äôve been primed to believe their product is probably a little cheaper than they initially thought, and that combined with their desire to get rid of it quickly means you‚Äôll likely get the sale at a discounted rate.</p><p>There are plenty of services that create quick burner e-mail addresses for you in a few seconds, making this easy. I won't link them here, but a quick Google search should suffice. In addition, there exist similar services for phone numbers as well, and if you <em>really</em> wanted to purchase the product, a few phone calls from friends or family members wouldn't hurt either.</p><p>Again, this method is not for everyone, and it can take a fair amount of time to coordinate if you‚Äôre doing it often. However, it works exceptionally well. One of the strongest psychological mechanisms human beings fall prey to is <em>social proof</em>‚Äîour willingness to make individual decisions based on those of the majority. Combine this strategy with email and the previous three rules, and you‚Äôre almost guaranteed to successfully negotiate on Craigslist every time.</p><h3 id="in-closing">In closing</h3><p>So, how do you negotiate on Craigslist? To briefly summarize the techniques we've gone over:</p><ol><li>Avoid professional Craigslisters that runn arbitrage schemes. Be wary of near-instant response times, multiple emails in quick succession, and sob stories‚Äîthese people will simply waste your time with sub-par offers.</li><li>When you find a deal that you're interested in pursuing, call the prospect rather than e-mailing them. Putting a voice to the name significantly improves your odds of building a good relationship, and allows you to haggle harder using the Ackerman method.</li><li>Speaking of the Ackerman bargaining method‚Äîuse it, and use it often. If buying a product, start by pitching at 65%, 85%, 95%, and 100% of your desired price. If selling, pitch at 135%, 115%, 105%, and 100% of your desired price. Between each pitch, include a soft rejection that implies that you really want the deal, but it's just not financially feasible for you right now. On the last offer, add a non-round number to show the other party that you're at your maximum, and if they still don't budge, throw in a non-monetary item to seal the deal.</li><li>If you're feeling unethical, send messages from multiple accounts to fabricate your desired price. You can do this with a variety of burner e-mail and phone number services to save yourself time (just be careful who you use this technique on, and mind your reputation).</li></ol><p>Many people have used these techniques to save thousands of dollars, trade their products at favorable rates, and ultimately get <em>off </em>Craigslist (and back to their lives) much faster than before. I sincerely hope you can, too.</p><p>Pat yourself on the back for finishing this article! You've now taken a deep dive into the nuanced techniques that make professional Craigslisters thousands of extra dollars per month. Don't worry if your first deal isn't absolutely incredible‚Äîlike anything else in life, Craigslist takes practice. You won‚Äôt be perfect the first time you try and negotiate, and it‚Äôs a constant learning experience that tests your social, mathematical, and financial skills.</p><p>But, like other things, if you persist, you will succeed.     function renderSubmenus() {
      const navItems = document.querySelectorAll('[data-nav] li')
      const subItems = document.querySelectorAll('.is-subitem')
      const mobileClass = "hidden pl-4 flex flex-col gap-0.5 py-1 px-1.5 basis-full"
      const desktopClass = "z-[50] min-w-[calc(100%_+_24px)] absolute flex flex-col gap-1 shadow-2xl border border-brd rounded-theme-xxs bg-bgr py-2 px-1.5 -ml-2 mt-11 opacity-0 invisible translate-y-1 transition-all"

      // Remove '-' signs
      subItems.forEach(item => {
        const itemName = item.querySelector('a span')
        itemName.innerText = itemName.innerText.slice(1)
      });

      // Add subitems in place
      let subMenu, hasItems
      navItems.forEach((item, index) => {
        if (item.classList.contains('is-subitem') && !navItems[index - 1].classList.contains('is-subitem')) {
          navItems[index - 1].classList.add('is-mainitem'); 
          navItems[index - 1].setAttribute('x-on:click.outside', 'closeSubmenus(event)')
        }
        subMenu = item.classList.contains('is-subitem') ? subMenu : document.createElement('ul');
        if (item.classList.contains('is-subitem')) { 
          subMenu.appendChild(item)
          subMenu.setAttribute('data-submenu', '')
          subMenu.getAttribute('id') === null ? subMenu.setAttribute('id', navItems[index - 1].getAttribute('data-slug')) : ''
          const menuType = subMenu.closest('nav').getAttribute('data-nav')
          subMenu.classList = menuType === 'desktop' ? desktopClass : mobileClass
        } else { 
          item.appendChild(subMenu)
        }
      });

      const dropdownMenus = document.querySelectorAll('[data-nav] .is-mainitem')
      const toggle = document.querySelector('[data-toggle-template]');

      dropdownMenus.forEach(menu => { 
        const toggleBtn = toggle.content.firstElementChild.cloneNode(true);
        toggleBtn.setAttribute('aria-controls', menu.getAttribute('data-slug'))
        menu.insertBefore(toggleBtn, menu.children[1]);
      });
    }
